# Dulith Divisekara — Personal Portfolio

Information Technology undergraduate portfolio showcasing networking, serverless edge systems, and embedded IoT projects. The site is a single-page HTML portfolio built with Tailwind CSS and vanilla JavaScript.

Key features

- Hero section with CLI-first resume link and copyable curl command
- About & education summary
- Skills grouped by software, networking, cloud, and embedded systems
- Featured projects: Cloudflare Workers edge resume, Arduino IoT framework, event leadership
- Contact form (Formspree) and direct contact links

Technologies

- HTML5, Tailwind CSS (via @tailwindcss/browser)
- JavaScript (vanilla)
- Hosted on Cloudflare Edge (edge workers for CLI resume)

Quick start (local)

1. Open index.html in a browser, or serve locally:
   python3 -m http.server 8000
2. Visit http://localhost:8000

Live / CLI resume

- CLI resume: curl https://cv.dulithdivisekara.workers.dev
- The portfolio is hosted on the Cloudflare Edge (see footer in index.html)
